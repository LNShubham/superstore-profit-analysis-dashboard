# Superstore Profit Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built using the Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, and product categories to support business decision-making.

---

## Business Objective

The goal of this project is to analyze retail sales data and identify trends, profitable regions, customer segments, and product categories.

---

## Tools Used

- Power BI
- Power Query
- Microsoft Excel
- DAX

---

## Dashboard Features

- KPI Cards
- Profit Trend Analysis
- Region-wise Profit Analysis
- Category-wise Profit Analysis
- Customer Segment Analysis
- Interactive Country Filter

---

## Key Insights

- North region generated the highest profit.
- Technology category was among the highest contributors to profit.
- Appliances and Phones were top-performing sub-categories.
- Corporate customers generated strong profits.
- Profit varied across quarters, highlighting seasonal patterns.
